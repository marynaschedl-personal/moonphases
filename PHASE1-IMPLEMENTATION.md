## 💡 Tips for Future Enhancement

1. **To add more animations**:
   - Use `@keyframes` in CSS, not JavaScript
   - Keep animations under 500ms
   - Use `transform` and `opacity` only

2. **To customize colors**:
   - Search for color hex codes (#00d4ff, #FFD700, etc.)
   - Update in both CSS and JavaScript

3. **To change timeline card count**:
   - Modify the `for (let i = -15; i <= 15; i++)` loop
   - Adjust timeline-card flex basis if needed

4. **To optimize performance**:
   - Use DevTools Performance tab to measure
   - Check Lighthouse score in DevTools
   - Run on slow 3G in DevTools to test
